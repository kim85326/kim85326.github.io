### 強制轉換

```
var x=5*2;
alert (x);      // ==>輸出 10
x="5"*"2";
alert (x);      // ==>字串5乘字串2 輸出 10
x=5*"2";
alert (x);      // ==>數字5乘字串2 輸出 10
x=5+"2";
alert (x);      // ==>數字5加字串2 輸出 52
x="5"+2;
alert (x);      // ==>字串5加數字2 輸出 52
```

### 存在
```

Boolean("")				// ==>false
```

所以

```
var a;

	console.log('exist');

	console.log('not exist');
```
