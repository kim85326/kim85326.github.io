* 當整個網頁( html, css, javascript )載入完才執行程式碼，避免找不到目標
```javascript=
$(document).ready(function(){
    程式碼
});
```	

* click 事件寫法
```javascript=
$("button").click(function(){
    程式碼
});        
```

```javascript=
$("button").on("click",function(){
    程式碼
});
```

程式碼
});
