# 求公因數/公倍數

### 利用輾轉相除法
![](http://scratch.gdps.ntpc.edu.tw/shu-xue-ti-ku/01guo-xiao-shu-xue/6nian-ji-shu-xue/6a0204qiuab-liang-shu-de-zui-da-gong-yin-shu-zhan-zhuan-xiang-chu-fa/2016-04-10_192314.png?attredirects=0)

### 演算法
```javascript=
// 最大公因數
function gcd(m,n){
    var remainder = 0;
    do{
        remainder = m % n;
        m = n;
        n = remainder;
    }while(remainder!==0);
    return m;
}
// 最小公倍數
function lcm(m,n){
    return (m*n) / gcd(m,n);
}

```

### 用遞迴的方式
```javascript=
function gcd(m,n){
    if(n) 
        return gcd(n,m % n);
    return m;
}
```
