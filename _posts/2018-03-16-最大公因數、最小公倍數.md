# 求公因數/公倍數

### 利用輾轉相除法
![](https://i.imgur.com/JpMXPXV.png)


### 演算法
利用javascript來做
```js
// 最大公因數
function gcd(m,n){
    var remainder = 0;
    do{
        remainder = m % n;
        m = n;
        n = remainder;
    }while(remainder!==0);
    return m;
}
// 最小公倍數
function lcm(m,n){
    return (m*n) / gcd(m,n);
}

```

### 用遞迴的方式
```js
function gcd(m,n){
    if(n) 
        return gcd(n,m % n);
    return m;
}
```
